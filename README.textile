h1. OneBody

"http://beonebody.org":http://beonebody.org

OneBody is open source software that connects community members on the web.

h2. Features

* Basic Church Management System (ChMS)
* Online Directory
* Groups (with email list management, attendance tracking, and more)
* Social Networking

...and more.

h2. Up And Running

<pre class="console"><code>
[sudo] rake gems:install
rake db:migrate
rake onebody:newuser # or rake onebody:load_sample_data
script/server
</code></pre>
...then browse to "http://localhost:3000":http://localhost:3000
